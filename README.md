# Gerenciador de Requisitos

Sistema web para cadastro e visualização de requisitos de software.

## Tecnologias usadas

- HTML
- CSS
- JavaScript
- PHP 

## Estrutura Principal

- index.php: tela inicial de cadastro de projeto.
- Interfacederequisitos/requisitos.html: cadastro de requisitos.
- Interfacederequisitos/deshboard/lista.html: dashboard/lista de requisitos.

## Funcionalidades

- Cadastro de requisito com validação de campos.
- Campos de requisito:
	- Título
	- Descrição
	- Classificação (funcional ou nao-funcional)
	- Prioridade (baixa, média, alta)
	- Status
- Persistência local no navegador (localStorage).
- Dashboard com:
	- Listagem de requisitos cadastrados
	- Filtro por tipo e status
	- Métricas (total, funcionais, nao-funcionais)
	- Exclusão de requisito

## Como Executar

Pré-requisito:
- PHP instalado e disponível no PATH.

Passos:
1. Abra o terminal na pasta do projeto: E:/Clone/Repmanager
2. Execute: php -S localhost:8000
3. Acesse no navegador: http://localhost:8000
